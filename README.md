# BUILD Desktop

Get up ready running Ubuntu or Fedora desktop

## How to run
```bash
export DEV_ENV=$(pwd) # I am considering you smart enough
./run
```

NOTE: if you are running on anyother OS than Fedora
do change line 44 in run

from:
$s "Fedora-w"
to:
$s "<option>"

the options you have are:
- Fedora-w
- fedora-x
- ubuntu-x
- ubuntu-w
